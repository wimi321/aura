# aura_core

Aura 的后端核心包，负责端侧角色扮演引擎的领域层、上下文编排与导入解析能力。

## 已完成能力

- 模型清单与热切换状态模型
- 统一 `AuraEngine` 会话入口
- 会话编排与默认角色扮演参数
- 世界书静默触发
- Whisper 指令高优先级注入
- 情绪标签剥离
- PNG 角色卡元数据解析
- ZIP 表情差分包解析
- JSON 预设解析
- 上下文占用阈值与摘要触发策略
- 会话文件仓库与内存仓库
- 模型目录文件仓库与内存仓库
- 可恢复 HTTP 模型下载器
- 设备能力到 Delegate 的运行时决策
- 纯 Dart 单元测试样例

## 仍需真实环境接入

- `flutter_litert` / LiteRT-LM 原生推理桥
- 真机设备 RAM / NPU / GPU 能力探测
- 真机音频帧采集与平台桥接
- iOS CoreML / Android NNAPI 的端到端联调验证
