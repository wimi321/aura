# 发给 Antigravity 的封面提示词

请你接手 Aura 内置剧情卡的第二轮正式封面升级。

这不是做“角色头像”或“聊天对象立绘”，而是做一个移动端互动剧情 App 的剧情入口海报。
核心要求是：用户一眼看上去就知道这是“点进去会直接进入某个剧情现场”的封面，而不是占位图、概念图、游戏启动页、二次元角色展示卡，或者 AI 味很重的花哨图。

我已经把技术规格、构图安全区、每张卡的剧情核心、必须出现的线索物件、以及不要踩的方向整理好了，你按这份 brief 出图即可：

- `docs/branding/antigravity_story_cover_brief.md`
- 参考当前待替换目标图：`docs/branding/antigravity-handoff-targets.png`

这次你主要替换这 8 张：
- `shadow-warden`
- `oath-arbiter`
- `last-train-keeper`
- `memory-smuggler`
- `night-prefect`
- `deskmate`
- `slayer-mage`
- `dungeon-arbiter`

其中优先级最高的 3 张是：
1. `memory-smuggler`
2. `oath-arbiter`
3. `dungeon-arbiter`

统一要求：
- 输出 `1600x2400` 竖版 PNG
- 不要在图里加标题、Logo、水印
- 底部要给 App 标题和简介留出可读空间
- 重点做剧情入口感、环境叙事感、人物与线索物件之间的关系
- 不要做成抽象符号海报
- 不要做成纯人像美术图
- 不要做成游戏化 splash 图
- 不要做成 AI 默认霓虹配色模板

你不需要被我绑定具体颜色，颜色和整体美术气质由你判断，但一定要避免一眼看上去像 AI 占位图。

质量目标：
- 像高端互动读物 / 限定剧集主视觉 / 精品剧情产品封面
- 不像聊天工具
- 不像 demo
- 不像工程项目配图

先从优先级最高的 3 张开始，统一气质确认后，再推进剩下 5 张。
